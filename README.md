# Arduino GPS Glider-Tracker

#Wiring
Arduino Nano (ATmega328P)

| GPS   | ARDUINO   | 
| :---  |     :---: |
| VCC   | +5V       |
| Rx    | D6        |
| Tx    | D5        |
| GND   | GND       |

| SD Card   | ARDUINO   | 
| :---      |     :---: |
|  CS       | D10       |
| SCK       | D13       |
| MOSI      | D11       |
| MISO      | D12       |
| VCC       | +5V       |
| GND       | GND       |

| DISPLAY   | ARDUINO   | 
| :---      |     :---: |
| SCL       | A5        |
| SDA       | A4        |

#Data Converting
    python3 process.py FILENAME
